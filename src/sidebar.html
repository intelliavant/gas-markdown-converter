Markdown Converter

Please select some texts and click the "Convert" button. It will convert the following Markdown syntax:

  • **bold** => bold
  • *italic* => italic
  • `monospace` => monospace

Also, a code block will be converted to a pre-like table. Note that the whole code block including the leading and trailing "```" must be in the same paragraph, i.e. new lines should be typed in with "Shift-Enter" instead of "Enter".

```
code block
code block
```

will be converted to:

code block
code block
